    ORDER

    • the function order() takes a vector and returns the indices that sorts that vector. For example
    
> x <- c(31, 4, 15, 92, 65)
> x
[1] 31  4 15 92 65
> sort(x)
[1]  4 15 31 65 92
> index <- order(x)
> x[index]
[1]  4 15 31 65 92
    • Rather than sort the vector, the function order gives us back the index that if used to index the vector, will sort it. In the example below, as you can see, we can sort the murders$abb following the index of murders$total
    
> x
[1] 31 4 15 92 65
> order(x)
[1] 2 3 1 5 4
> index <- order(murders$total)
> murders$abb[index]
 [1] "VT" "ND" "NH" "WY" "HI" "SD" "ME" "ID" "MT" "RI" "AK" "IA" "UT" "WV" "NE" "OR" "DE"

    MAX AND WHICH.MAX

    • If we want to get the max value of a column we can use function max(). For example
    
> max(murders$total)
[1] 1257
    • The function which.max() gives us the index where this max number resides. For example, If we want to find the state with highest total murders we can do as below.
    
> i_max <- which.max(murders$total)
> i_max
[1] 5
> murders$state[i_max]
[1] "California"

    MIN AND WHICH.MIN

    • If we want to get the min value of a column we can use function min(). For example:
    
> min(murders$total)
[1] 2
    • The function which.min() gives us the index where this min number resides. For example, If we want to find the state with lowest total murders we can do as below.
    
> i_min <- which.min(murders$total)
> i_min
[1] 46
> murders$state[i_min]
[1] "Vermont"

    RANK

    • Then function rank() give you a vector with the rank of first entry, ... last entry. For example
    
> x <- c(31, 4, 15, 92, 65)
> x
[1] 31  4 15 92 65
> rank(x)
[1] 3 1 2 5 4

    SUMMARY

    • The function sort() sorts a vector in increasing order.
    • The function order() produces the indices needed to obtain the sorted vector, e.g. a result of 2 3 1 5 4 means the sorted vector will be produced by listing the 2nd, 3rd, 1st, 5th, and then 4th item of the original vector.
    • The function rank() gives us the ranks of the items in the original vector.
    • The function max() returns the largest value, while which.max() returns the index of the largest value. The functions min() and which.min() work similarly for minimum values.
    
library(dslabs)
data(murders)
sort(murders$total)

x <- c(31, 4, 15, 92, 65)
x
sort(x)    # puts elements in order

index <- order(x)    # returns index that will put x in order
x[index]    # rearranging by this index puts elements in order
order(x)

murders$state[1:10]
murders$abb[1:10]

index <- order(murders$total)
murders$abb[index]    # order abbreviations by total murders

max(murders$total)    # highest number of total murders
i_max <- which.max(murders$total)    # index with highest number of murders
murders$state[i_max]    # state name with highest number of total murders

x <- c(31, 4, 15, 92, 65)
x
rank(x)    # returns ranks (smallest to largest)
