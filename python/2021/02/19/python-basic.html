    HISTORY

    • Python is a general-purpose interpreted, interactive, object-oriented, and high-level programming language. It was created by Guido van Rossum during 1985- 1990. Like Perl, Python source code is also available under the GNU General Public License (GPL).

    • Python is a high-level, interpreted, interactive and object-oriented scripting language. Python is designed to be highly readable. It uses English keywords frequently where as other languages use punctuation, and it has fewer syntactical constructions than other languages.

    LIST

    • List is a sequence of mutable values. In python we can define a list of elements as in the example below. Then, to add new element into a list we can use append() and to sort a list we can use sort()
    #create an empty list
my_list = []

#create a list of names
names = ["Harry", "Ron", "Tom"]
print(names)

#add a new element into a list with append()
names.append("Duc")
print(names)

#sort list names with sort()
names.sort()
print(names)

    <!-- result -->
['Harry', 'Ron', 'Tom']       
['Harry', 'Ron', 'Tom', 'Duc']
['Duc', 'Harry', 'Ron', 'Tom']

    SET

    • Set is a collection of unique values. In python, we use set() to create an empty set then use add() to add new element into a set and remove() to remove an element of the set. Moreover, we can use len() to count the number of elements of list, set…
    #create an empty set
s = set()

# add elements into set
s.add(1)
s.add(2)
s.add(3)

s.add(3)
s.remove(1)
print(s)

print(f"the set has {len(s)} elements.")

    <!-- the result of terminal will be showed -->
{2, 3}
the set has 2 elements.

    PYTHON ANONYMOUS/LAMBDA FUCNTION

    • In python, an anonymous function is a function that is defined without a name.
    • While normal functions are defined using the def keyword in Python, anonymous functions are defined using the lambda keyword. Hence, anunymous functions are also called lambda functions.

      Sysntax of Lambda function in python

      lambda arguments: expression/body

    • Let’s see the example below:

    • Example 1 
      # normal function with input parameter x and return x
def identity(x):
  return x

      # lambda function with input parameter x and return x
lambda x: x
    • Example 2 
      # normal function with input parameter x, y 
# and return x * y
def identity(x, y):
  return x * y

      # lambda function with input parameter x, y 
# and return x * y
lambda x, y: x * y
    • Example 3
    # We have a list of dict (list of 3 people) and 
# we want to sort the list following name of person
people = [
    {"name": "Harry", "house": "Gryffindor"},
    {"name": "Cho", "house": "Ravenclaw"},
    {"name": "Draco", "house": "Slytherin"}
]

# Normal function that returns value 
# of name (Harry or Cho or Draco)
def f(person):
    return person["name"]

# We sort the list following key is name of person
people.sort(key=f)
print(people)

    # We have a list of dict (list of 3 people) and 
# we want to sort the list following name of person
people = [
    {"name": "Harry", "house": "Gryffindor"},
    {"name": "Cho", "house": "Ravenclaw"},
    {"name": "Draco", "house": "Slytherin"}
]

# We sort the list following key is name of person
# Using lambda function we have input parameter is person
# and we return the value of name in person
people.sort(key=lambda person: person["name"])
print(people)

    <!-- the result of both cases -->

[{'name': 'Cho', 'house': 'Ravenclaw'}, {'name': 'Draco', 'house': 'Slytherin'}, {'name': 'Harry', 'house': 'Gryffindor'}]

    EXCEPTIONS

    • In Python, We can use try and except to handle exceptions. Let’s see the example below.
    import sys

# We will check the input value from the terminal
# If the input value is not int type the program will be exited
try:
    x = int(input("x: "))
    y = int(input("y: "))
except ValueError:
    print("Error: Invalid input!")
    sys.exit(1)

# We will check the callculation of x / y
# if y is 0 the program will be exited
try:
    result = x / y
except ZeroDivisionError:
    print("Error: Can not divide by 0.")
    sys.exit(1)

print(f"{x} / {y} = {result}"


    <!-- exception with input value of x is not int -->
x: hello
Error: Invalid input!

<!-- exception with input value of y is 0 -->
x: 5
y: 0
Error: Can not divide by 0.

<!-- result with no exception -->
x: 5
y: 5
5 / 5 = 1.0
