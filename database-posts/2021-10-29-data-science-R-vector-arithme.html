    Vector Arithmetic

    • As we study in the previous post, the state California has the most murders and the most population. So Is this state will be the most dangerous state?
    
> murders$state[which.max(murders$population)]
[1] "California"
> max(murders$population)
[1] 37253956

    • As we can see the population of California is 37253956 so it is unfair if we compare California to other states. So What we really should be computing is the murders per capita.

    • In R, arithmetic operations on vectors occur element-wise. For example: We a vector of heights and we want to compute how many inches taller or shorter than the average 69 inch. If we want to view the heights with cm we can *2.54.

    
> heights <- c(69,62,66,70,70,73,67,73,67,70)
> heights * 2.54
 [1] 175.26 157.48 167.64 177.80 177.80 185.42 170.18 185.42 170.18 177.80
> heights - 69
 [1]  0 -7 -3  1  1  4 -2  4 -2  1

    • One more thing, if we have 2 vector on the same length so we can + - * / 2 vector and every entry them will be calculated respectively.

    SUMMARY

    • In R, arithmetic operations on vectors occur element-wise.
    
# The name of the state with the maximum population is found by doing the following
murders$state[which.max(murders$population)]

# how to obtain the murder rate
murder_rate <- murders$total / murders$population * 100000

# ordering the states by murder rate, in decreasing order
murders$state[order(murder_rate, decreasing=TRUE)]
